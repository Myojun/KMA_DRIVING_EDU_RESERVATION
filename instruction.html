📖 예약시스템 이용 가이드

운전교육 예약시스템을 쉽고 빠르게 이용하는 방법을 안내합니다.

1 예약방법

1

예약하기 페이지 접속

홈에서 예약하기 버튼을 클릭하거나, 우측 상단 메뉴에서 예약하기를 선택합니다.

2

날짜 선택

캘린더에서 원하는 교육 날짜를 선택합니다. 선택한 날짜의 교육 일정이 실시간으로 표시됩니다.

3

시간대 및 교육 선택

원하는 시간대(08시, 10시, 13시, 15시)와 교육(기능, 도로기초, 도봉)을 선택합니다.
미개설로 표시된 시간대는 예약할 수 없습니다.

4

개인정보 입력

이름, 학년, 중대, 전화번호를 정확히 입력합니다.
전화번호는 예약 조회 시 필요하므로 정확히 입력해주세요.

5

예약하기 버튼 클릭

모든 정보를 입력한 후 예약하기 버튼을 클릭하면 예약이 완료됩니다.

💡 예약 팁

• 당일 예약 시 예약 반영되지 않으니 미리 확인하고 신청하기 바랍니다.

• 동일한 날짜에는 한 번만 예약 가능합니다.

예약하기 화면 예시

2 예약조회방법

1

예약조회 페이지 접속

홈에서 예약조회 버튼을 클릭하거나, 우측 상단 메뉴에서 예약조회를 선택합니다.

2

이름과 전화번호 입력

예약 시 입력한 이름과 전화번호를 정확히 입력합니다.
두 정보가 모두 일치해야 조회가 가능합니다.

3

조회하기 버튼 클릭

조회하기 버튼을 클릭하면 예약 내역이 테이블 형태로 표시됩니다.

⚠️ 주의사항

이름과 전화번호가 정확히 일치해야만 조회가 가능합니다.

예약조회 화면 예시

3 예약취소방법

1

예약조회 페이지에서 예약 확인

먼저 예약조회 페이지에서 본인의 예약을 조회합니다.

2

취소 버튼 클릭

조회된 예약 내역의 우측에 있는 [취소] 버튼을 클릭합니다.

3

취소 확인

취소 확인 메시지가 나타나면 확인을 클릭합니다. 예약 상태가 "취소됨"으로 변경됩니다.

⚠️ 취소 시 주의사항

• 교육 반영 문자 수신 후 취소 시에는 교관에게 연락해야합니다.

• 취소 후 동일한 날짜에 다시 예약하려면 새로 예약해야 합니다.
